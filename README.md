# 퐁퐁이 게임 — 요구사항

## 공통 사항
- 개발 언어: **HTML + JavaScript** (웹 게임, 브라우저에서 바로 실행)
- 개발 환경: WSL 내 우분투
- 실행 방법: HTML 파일 하나를 크롬에서 열면 바로 플레이 (설치 불필요)

## 기본 정보
- **게임명**: 퐁퐁이 (Pongpong)
- **참고 캐릭터**: 별의 요정 커비 (둥근 핑크 캐릭터)
- **팀원**: 김지유(7살), 김종진
- **한 줄 설명**: 퐁퐁이가 박스를 밟고 위로 올라가며 별을 모으고, 별을 다 모으면 열쇠 든 사람에게서 열쇠를 받아 문을 열고 클리어하는 점프 게임

---

## 게임 목표 (승리 조건)
1. 퐁퐁이가 박스를 밟고 **위로 올라간다**.
2. 올라가는 길에 놓인 **별 ⭐ 3개**를 모두 모은다.
3. 별 3개를 다 모으면, 맨 위에 있는 **사람 🧍이 열쇠 🔑를 준다**.
   - 별을 다 모으기 전에는 사람이 "별 더 모아와~" 하며 열쇠를 주지 않는다.
4. 열쇠를 받으면 **문 🚪이 열리고 → "클리어! 🎉"**.

## 패배 조건
- 박스에서 떨어지면(추락) **하트 ❤️ 1개 감소** + 아래쪽에서 다시 시작.
- 하트 **5개를 모두 소진하면 → 게임 오버** → "다시 하기" 버튼.

---

## 등장 요소
| 요소 | 설명 |
|---|---|
| 🩷 퐁퐁이 | 주인공. 커비 같은 둥근 핑크 캐릭터. 좌우 이동 + 점프. 통통 튀는 느낌. |
| 📦 박스 | 밟고 올라가는 발판. 여러 개가 계단처럼 위로 배치. |
| ⭐ 별 | 총 3개. 가는 길에 놓여 있고, 다 모아야 열쇠를 받을 수 있다. |
| 🧍 사람 | 맨 위 목표 지점. 별 3개를 다 모으면 열쇠를 준다. |
| 🚪 문 | 사람 옆에 있다가, 열쇠를 받으면 스르륵 열린다. |
| ❤️ 하트 | 목숨. 5개로 시작. 추락하면 1개씩 감소. |

## 조작 방법 (클릭/터치만)
- 화면에서 **누른 쪽(왼쪽/오른쪽)으로 퐁퐁이가 이동**한다.
- 박스 앞에서 누르면 **자동으로 통통 점프**한다.
- → 7살 아이가 마우스 또는 손가락 하나로 조작 가능.

## 화면 · 느낌
- 밝고 알록달록한 색 (파란 하늘, 초록 바닥, 알록달록 박스).
- 퐁퐁이가 점프할 때 살짝 찌그러졌다 펴지는 "뿅!" 효과.
- 별 먹을 때 반짝 효과 + 소리.
- 클리어하면 "딩동댕! 🎉" 축하 화면.
